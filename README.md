# Intelli Grade API

This is a basic FastAPI setup.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the server:
   ```bash
   uvicorn main:app --reload
   ```

3. Open your browser to http://127.0.0.1:8000/docs for the interactive API documentation.
